# @saudi-mcp/shared

Shared building blocks for [saudi-mcp-servers](https://github.com/): a small HTTP client with retry/backoff and typed error mapping, an `ApiError` taxonomy, and conservative Arabic text helpers (diacritic stripping, letter normalization, Arabic-Indic digit conversion) used for matching — never for display.

Internal package; published so individual servers can depend on it.
