# saudi-mcp-servers

خوادم [Model Context Protocol](https://modelcontextprotocol.io) (MCP) مفتوحة المصدر للواجهات البرمجية والخدمات السعودية — لتمكين وكلاء الذكاء الاصطناعي من التعامل مع الأنظمة الحكومية والتجارية السعودية بشكل أصيل.

> English: see [README.md](./README.md)

## لماذا

وكلاء الذكاء الاصطناعي يتصلون اليوم بمئات الخدمات الغربية عبر MCP، بينما لا تكاد توجد خدمة سعودية لها خادم MCP مفتوح. هذا المستودع يسدّ الفجوة، بدءاً بالأنظمة التي تتعامل معها كل منشأة سعودية.

**مبدأ التصميم:** أسماء الأدوات وأوصافها بالإنجليزية (لأن استدعاء الأدوات بالعربية ما زال هشّاً)، بينما **البيانات** المُعادة تبقى بالعربية. هكذا يختار الوكيل الأداة بثبات، ويحصل المستخدم على نتائج عربية أصيلة.

| الحزمة | الخدمة | الحالة |

| [`@saudi-mcp/wathq`](./packages/wathq) | واثق — السجل التجاري (وزارة التجارة) | ✅ الإصدار 0.1 |
| `@saudi-mcp/zatca` | هيئة الزكاة — الفوترة الإلكترونية (فاتورة) | 🔜 قادم |

## البدء السريع (واثق)

```bash
npx @saudi-mcp/wathq
```

عيّن مفتاح واثق:

```bash
export WATHQ_API_KEY="مفتاحك_هنا"
```

أو جرّبه **بدون مفتاح** ببيانات تجريبية:

```bash
WATHQ_MOCK=true npx @saudi-mcp/wathq
```

## ملاحظة حول الوصول

الخادم هو **الموصِّل** مفتوح المصدر. لاستخدامه مع بيانات حقيقية تحتاج اشتراكك الخاص في [واثق](https://developer.wathq.sa) ومفتاحك. التطبيق المجاني لمدة 30 يوماً يكفي للبيئة التجريبية أثناء التطوير.
